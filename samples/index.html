DC D3 Crossfilter and Other D3 Compatible Samples

Welcome to the world of DC.JS, D3.JS , Other D3 based visualization frameworks !!
Use the "Samples" drop down menu or click on the sample description below to explore the samples

Some of the samples like the Earthquake dashboard use live data GET calls which will cause a CORS exception.Running those specific samples on the browser will require CORS disabled in browsers to work. In Chrome you can use --allow-file-access-from-files --disable-web-security options for the same. As a fall back if CORS is not enabled it relies on an offline json dataset.

Vehicle Registration for Indian cities - dashboard

The data format has been taken from data.gov.in and provides a dashboard view of the registered motor vehicles in India from 2001 to 2011. The dashboard is looking at cities which have more than a million vehicle registrations.This uses melt.js, crossfilter,d3 and dc for the visualization. Used dc, crossfilter, d3 and melt.js for the data processing and visualization.

US population density by state capitals

This uses the DC.js choropleth charts to map US state wise population with color coding of red to green to indicate the lower to high density of population. This has been inspired by the wonderful sample provided for US Venture Capital Landscape 2011 (http://dc-js.github.io/dc.js/vc/)
